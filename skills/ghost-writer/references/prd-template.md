# PRD Template

## 1. Vision & Mission
*   **Vision**: The long-term aspirational goal.
*   **Mission**: What the product is doing now to achieve the vision.

## 2. Problem Statement
*   Clearly define the pain points being solved.
*   Identify the status quo and why it's insufficient.

## 3. Target Audience
*   Who are the primary users?
*   What are their specific needs/desires?

## 4. Core Features
*   **Feature A**: Description and value proposition.
*   **Feature B**: Description and value proposition.
*   ...

## 5. User Workflow
*   Step-by-step description of how a user interacts with the product.

## 6. Technical Constraints & Assumptions
*   Technology stack choices.
*   Known limitations.

## 7. Success Metrics
*   How will we know the product is successful? (KPIs).

## 8. Open Questions / Puzzles
*   Unresolved issues or areas needing further research.
